Acerca de mi

Soy venezolano, profesional como desarrollador de software desde 2016, me títule como Licenciado en Computacion y T.S.U en Informática, amante de la tecnología y de encontrar soluciones con el uso de algoritmos, mi principal virtud es la responsabilidad a la hora de cumplir con los objetivos establecidos y lograr solucionar los problemas de los clientes otorgando un trabajo de calidad.

Como FrontEnd he trabajado en el diseño y desarrollo de paginas Web (HTML, CSS, JavaScript) implementado ademas SEO Orgánico y herramientas como Google Analytics, API GRAPH de Facebook, Google Maps Api, actualmente estoy fortaleciendo mis conociemientos en React Js y Angular

Como BackEnd he participado en proyectos de Web Scraping con java pero principalmente me he dedicado al desarrollo y mantencion de Microservicios (Api Rest) utilizando Spring Boot, asi como la implementacion de Api Proxies con ApiGee de Google también poseo solidos conocimientos en PHP

A nivel de Base de Datos he trabajado ampliamente motores relacionales como MySQL y PostgreSQL (Postgis para manejos de datos Geo-Espaciales), aunque tambien tengo conocimientos basicos de base de datos basadas en documentos

A nivel de gestion de proyectos he tenido la oportunidad de ejercer como Scrum Master utilizando Jira para la planificacion de tareas y actividades

Conociemientos Basicos en Jenkins y Kubernetes para la integracion CI/CD, Fortify y Sonar como herramientas de debug de errores y/o vulnerabilidades de seguridad

Trabajo constantemente con Control de Versiones (Git), Docker (Orquestacion de contenedores con docker-compose) en la implementacion de servicios cloud en las plataformas de AWS y GCP junto con herramientas para pruebas como Postman y JMeter.
